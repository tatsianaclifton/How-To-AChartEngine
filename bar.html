Bar Chart

The process of creating a bar chart does not differ a lot from creating other types of charts. Create a new activity that will display the bar chart. For that, create two new files: activity_bar.xml and BarActivity.java. Include the new activity into the AndroidManifest.xml file:

<activity android:name=".BarActivity" />

The activity_bar.xml is the same as for the line chart.

The BarActivity.java is very similar for other charts we have created. However, while creating the graphical view we also need to specify the type. There are some options such as DEFAULT, STACKED, HEAPED. We will choose DEFAULT:

GraphicalView chart = ChartFactory.getBarChartView(BarActivity.this, dataset, mRenderer, BarChart.Type.DEFAULT);
