How-To Use AChartEngine for Android Applications

Getting Started

Introduction

AChartEngine is a charting library for Android applications. It supports a wide range of chart types. The AChartEngine official web site can be found here. This guide will demonstrate how to create an Android application that utilizes the AChartEngine library.

Add Library to Project

Navigate to the download section from the official home page of AChartEngine. Open the nightly build link that has the most current download. You should see the following screen:

Download the library file which has the .jar extension.

In Android Studio, create a new project. Go to File -> New -> New Project. In the Application name field, type the project's name (for example, Chart). Press Next, choose Phone and Tablet and select the Minimun SDK. On the next screen choose an Empty Activity and press Next. Leave the activity name as default MainActivity and press Finish.

Now we need to add the AChartEngine library to the created project. Right click on the app and select New -> Module.

Select Import .JAR/AAR Package.

On the next screen, for File Name, go to where you saved the AChartEngine library .jar file and select it. Then press Finish.

Now you need to add this library as dependency into your project. For that, right click on the app and select Open Module Setting. You will see:

Select Dependencies, then press the green plus sign and Module dependency.

Select the module you have added with the AChartEngine library and choose OK.

The AChartEngine library was included in the project and we can start creating various charts.

Creating Main Activity

First, we create the main activity of our app. This activity will have 7 buttons for each chart type. If a user clicks on a button, the relevant chart will be displayed.

Open activity_main.xml and create 7 buttons. For example, the button that will open the line chart has the following code: 

<Button
    android:layout_width="150dp"
    android:layout_height="wrap_content"
    android:text="Line Chart"
    android:id="@+id/line"
    android:onClick="drawLineChart"
    android:background="#9d67c1"
    android:layout_below="@+id/textView"
    android:layout_alignParentLeft="true"
    android:layout_alignParentStart="true"/>

We assiged to it the onClick method drawLineChart. This method will open the new activity that displays the line chart. Using the same steps, create the other buttons. You should receive similar results to the below image:

In MainActivity.java create methods that will be called onClick by each button. Each method will open the relevant activity with the appropriate chart. For the line chart, the code is the following: 

public void drawLineChart(View view){
        Intent intent = new Intent(MainActivity.this, LineActivity.class);
        startActivity(intent);
    }

In the next sections, we will see how to create different kinds of charts. Additionally, we will see how to customize them.
